# らくらく告知

「らくらく告知」は、お店の営業予定をカレンダーに登録し、SNSでのお知らせを準備できるWebアプリです。スマートフォンから使えます。

## できること

- 日付をタップして、営業日、お休み、変則営業を登録
- SNS用の正方形画像と、印刷用のA4画像を保存
- 季節に合わせた投稿文を作成
- Instagram、X、Threads、Facebookから投稿先を選択

## 使い方

1. 今月または来月のカレンダーを開きます。
2. お休みや変則営業の日をタップし、必要があれば特記事項を入力します。
3. 内容を保存し、カレンダー画像と投稿文を確認します。
4. 投稿先のSNSを開き、画像を添付して投稿します。

## データの保存

登録した内容と設定は、使用中のブラウザに保存されます。別の端末やブラウザとは自動で共有されません。

## AIリライト

特記事項を読みやすい文章に整えるAI機能です。利用には確認コードが必要です。利用方法や設定については、アプリ開発者へ直接お問い合わせください。

## ローカルで確認する

静的ファイルを配信できるWebサーバーで、プロジェクトのルートを開いてください。`file://` で直接開くと、一部のPWA機能が動作しない場合があります。
